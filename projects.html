Snowball Bash

Pygame similar to space invaders where you are throwing snowballs to keep the bullies away. Players use the right and left arrow keys to move and use spacebar to throw snowballs. If the enemies reach you, it's GAME OVER. Good luck players!

This is game is subject to change with more updates/features in the future.

Technologies: Python/pygame

myTunes

A playlister site where users can create and post playlists with videos. Features include creating a playlist with an author, description. After the playlist is created, videos can be added from sites like Youtube. Log in to create and view.

Technologies: Python, HTML, CSS, Django

Gohub

Web scraper of GitHub's Go trending page. When running, a JSON file will be generated with the top repos on github at the time when the code was run. One line of code can be modified to scrape top repos that are of a different language.

Technologies: Go
